# DDAEnforcer

An Android toolkit for Demand Deposit Account (DDA) enforcement and compliance checking.

## Overview

DDAEnforcer is a mobile application designed to enforce and validate Demand Deposit Account rules and compliance requirements directly from an Android device.

## Contents

| File | Description |
|------|-------------|
| `DDA Enforcer.apk` | Android application package |

## Requirements

- Android device (compatible with the APK's target SDK)
- Sufficient permissions for account validation operations

## Usage

1. Transfer `DDA Enforcer.apk` to your Android device
2. Enable installation from unknown sources in device settings
3. Install and launch the application
4. Follow in-app prompts for DDA enforcement workflows
